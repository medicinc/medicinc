# ASSET_PROVENANCE

Track each non-code asset used in the project with source and rights.

## Required fields per asset
- `path`: project-relative file path
- `type`: image / audio / svg / video / icon
- `source`: self-created / Adobe AI / Pixabay / other
- `license`: applicable license name/version
- `proof`: URL, invoice, download receipt, or generation log
- `commercial_use`: yes/no/conditional
- `notes`: restrictions, attribution obligations

## Initial notes
- Many UI graphics were generated during development chats and via Adobe AI.
- Audio files are mostly from Pixabay and should be re-verified per file prior to launch.
- Keep this file updated when adding or replacing assets.
